# Daniele Frisanco - Personal Portfolio

This repository contains the source code for my personal portfolio website, built with Jekyll and hosted on GitHub Pages.

**Live Site:** [https://danielefrisanco.github.io/](https://danielefrisanco.github.io/)

## Overview

This is a simple, clean, and responsive single-page portfolio designed to showcase my professional experience, skills, and contact information. It is built using the `minima` Jekyll theme and customized with additional styling and features.

## Running Locally

To run the site on your local machine, you'll need to have Ruby and Bundler installed.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/danielefrisanco/danielefrisanco.github.io.git
    cd danielefrisanco.github.io
    ```

2.  **Install dependencies:**
    This command installs all the required gems from the `Gemfile`.
    ```bash
    bundle install
    ```

3.  **Run the Jekyll server:**
    This command builds the site and serves it on a local development server.
    ```bash
    bundle exec jekyll serve
    ```

4.  **Preview the site:**
    Open your browser and navigate to `http://127.0.0.1:4000`. The site will automatically regenerate whenever you make changes to the source files.

## Technologies Used

*   **Jekyll:** A simple, blog-aware, static site generator.
*   **Minima:** The default Jekyll theme, which has been extended with custom includes and styles.
*   **GitHub Pages:** Hosting for the static website.
